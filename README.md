## 📌 Project Description

**AI Video Tool Selector** is an interactive Streamlit dashboard designed to help users discover and choose the best AI video generation tools based on pricing, ratings, and key features.  

With the rapid growth of AI-powered video platforms, this tool simplifies decision-making by providing a clear, visual comparison of popular tools in one place.

---

## 🚀 Features

- 🔍 **Smart Filters**
  - Filter by price range (Free / Freemium / Paid)
  - Filter by use case (Text-to-Video, Editing, Avatar, etc.)
  - Minimum rating slider

- 📊 **Interactive Data Table**
  - Compare tools by features, pricing, and ratings
  - Search by tool name or keyword

- 📈 **Visual Insights**
  - Bar chart: Average rating by pricing type
  - Scatter plot: Price vs Rating distribution

- ⚡ **Recommendation-Oriented Exploration**
  - Quickly identify the best tool based on your needs
  - Highlight affordable and high-performing options

- 📌 **Key Metrics Dashboard**
  - Total available tools
  - Average rating
  - Cheapest paid option
